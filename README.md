# 🤖 AI Career Coach

AI-powered resume analysis platform built with Generative AI.

## Features

- AI Resume Analyzer
- ATS Score Dashboard
- Skill Radar Visualization
- Resume vs Job Description Matching
- Interview Trainer
- Resume Chatbot
- Resume Knowledge Assistant (RAG)

## Tech Stack

- Python
- Streamlit
- Ollama (phi3)
- Plotly
- Generative AI
